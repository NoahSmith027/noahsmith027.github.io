While a lot of audio visualizers are abstract, I wanted to make one that looked like a more natural environment and visually modify the environment with music. It also switches automatically or manually between day and nighttime visual styles that have a few differences between them to prevent repetitiveness. The project was made using Canvas and Materialize.

This project is not perfect, though, and has some problems I need to fix:

  • extra black line on nighttime's moon: caused by an unclosed draw
  • canvas is off-center
  • treble and bass sliders do not properly function as they visually appear
  • visual modifier controls function but heavily slow down the visualizer, likely an efficiency issue
